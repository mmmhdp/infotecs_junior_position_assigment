# Задание 1

## Описание

Есть микросхема EEPROM-памяти с интерфейсом SPI (например, **25LC040A**).

Имеется драйвер, позволяющий управлять линиями микросхемы через переключение битов (*bit-banging*).

Необходимо реализовать **C++ класс (или набор классов)** для работы с микросхемой посредством данного драйвера.

## Функциональные требования

Требуется реализовать:

- Запись данных по адресу:
  - одного бита
  - одного байта
  - массива байт
- Чтение данных по адресу:
  - одного бита
  - одного байта
  - массива байт

Также необходимо описать, **что изменится**, если потребуется реализовать драйвер для работы с **NOR-памятью**
(например, **W25Q128**).

## Документация

- Все методы, константы, данные, перечисления и т.п. должны быть **документированы**
- Предпочтительный формат документации — **Doxygen**

## Требования к коду

- Декларации классов должны быть **синтаксически корректными**
- Код должен **компилироваться** (реализация может быть частичной, но корректной)

Готовое задание должно быть передано:

## Требования к структуре проекта

- Проект должен находиться в **отдельной папке**
